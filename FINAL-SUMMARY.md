## 📝 **For Users**

### **Quick Start**
```bash
# Install
npm install -g @rvegajr/aidb

# Verify
aidb --version  # 1.0.1

# Connect to database
aidb connect mydb --connection-string "mysql://user:pass@localhost:3306/db"

# View schema
aidb schema mydb

# Analyze relationships
aidb relationships mydb
```
